/**
需求详情：

软件名称：短信自动回复
软件用途：低成本网站验证码解决方案
软件使用环境：Android
功能：当安装此软件的手机收到包含指定文字的短信时，将来电手机号的后三位数按指定公式计算后自动回复，
例如：我设置关键字“验证码”  公式“x7+14” 
当我手机收到13764565987发来含有“验证码”三个字的短信时将987x7+14的结果6923发送回去
开机自动运行
*/
